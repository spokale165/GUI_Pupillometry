<h1>Pupillometry GUI </h1>
<br>
To run the web application on local host 
<br>
<br>
Clone the repo and follow the below mention commands on command line/terminal - 
<br>
<br>

<b>npm install</b>
<br>
<br>
<b>npm start </b>

<br>
<br>
This will start web application on the local host!!
